# 달빛선생 안전 카피 가이드

달빛선생의 해석은 삶의 흐름을 참고하기 위한 구조 해석입니다.  
건강, 투자, 법률, 위기상황 판단은 사주 해석과 분리해 안내합니다.

## 금지 표현

- “반드시 망한다”
- “대흉이니 피해야 한다”
- “질병이 생긴다”
- “치료된다”
- “이 종목을 사라”
- “소송에서 이긴다”

## 허용 표현

- “참고 흐름으로만 보세요”
- “생활 리듬 관리 관점에서 읽습니다”
- “투자 판단은 별도 전문 기준이 필요합니다”
- “법률 판단은 전문가 상담이 필요한 영역입니다”
- “위기상황은 사주 해석보다 즉각적인 도움 연결이 우선입니다”

## 화면 적용 원칙

1. 공포성 길흉 표현보다 생활 리듬, 선택 기준, 관계 구조 중심으로 설명합니다.
2. 건강 관련 표현은 진단이나 치료가 아니라 컨디션·휴식·생활 관리 차원으로만 제한합니다.
3. 투자 관련 표현은 매수·매도 권유 대신 지출 관리, 공격성 조절, timing 참고 흐름으로 바꿉니다.
4. 법률 관련 표현은 결과 단정 대신 전문가 상담 필요성을 먼저 안내합니다.
5. 위기상황은 해석보다 즉각적인 안전 확보와 도움 연결이 우선이라는 문장을 유지합니다.

## 권장 톤

- 정중하지만 과장하지 않습니다.
- 불안을 키우기보다 선택 기준을 차분히 정리합니다.
- “반드시”, “무조건”, “대흉” 같은 단정어는 기본 UI 카피에서 피합니다.

## 구현 메모

- SAFE_REDIRECT 같은 기존 감지/전환 로직은 유지합니다.
- UI에서는 `SafetyNotice` 공통 컴포넌트로 같은 메시지를 반복 가능하게 유지합니다.
- 실제 정책 확대 시 의료·법률·투자·위기상황 키워드 점검 체크리스트와 함께 운영합니다.
