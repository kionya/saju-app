# 달빛선생 안전 카피 가이드

## 목적

달빛선생의 해석은 삶의 흐름을 참고하기 위한 구조 해석입니다.  
의료, 법률, 투자, 위기상황처럼 전문 판단과 즉각적인 도움이 필요한 영역에서는 공포성 단정 표현을 피하고, 별도 전문 기준과 도움 연결을 우선합니다.

## 금지 표현

- “반드시 망한다”
- “대흉이니 피해야 한다”
- “질병이 생긴다”
- “치료된다”
- “이 종목을 사라”
- “소송에서 이긴다”

## 허용 표현

- “참고 흐름으로만 보세요”
- “생활 리듬 관리 관점에서 읽습니다”
- “투자 판단은 별도 전문 기준이 필요합니다”
- “법률 판단은 전문가 상담이 필요한 영역입니다”
- “위기상황은 사주 해석보다 즉각적인 도움 연결이 우선입니다”

## 화면 적용 원칙

- 결과 화면: 해석의 분위기를 정리하되 단정형 길흉 문구는 피합니다.
- 대화 화면: SAFE_REDIRECT 로직은 그대로 두고, UI 안내로만 안전 기준을 덧붙입니다.
- 샘플 리포트: 상품 구조를 보여주는 페이지이므로 실제 진단처럼 읽히는 표현을 피합니다.
- 멤버십/결제 인접 화면: 가격과 상품 설명에 불안을 자극하는 카피를 넣지 않습니다.

## 권장 톤

- 차분하고 직접적일 것
- 경고보다 안내에 가까울 것
- “절대”, “무조건”, “반드시” 같은 단어를 남용하지 않을 것
- 건강, 투자, 법률, 위기상황은 사주 해석과 분리해서 표현할 것
